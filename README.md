# Friend Location Tracking App (Android)

## Overview
A real-time location tracking app built using Android Studio. Uses Firebase and Google Maps API for live updates.

## Features
- Live GPS tracking
- Real-time location sharing
- User authentication
- Interactive map interface

## Technologies Used
- Android Studio
- Java
- Firebase Realtime Database
- Google Maps API

Developed the full app including backend integration, UI, and testing.
